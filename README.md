#SpaceX Past Launches Tracker
This is an example Android application using MVVM design pattern to show the list of SpaceX Past Launches and Launch Details.
Written in 100% Kotlin

Concepts used:
1. MVVM Architecture
2. Android JetPack Components
    Navigation Component
    Paging
3. Kotlin
4. Dagger Hilt Dependency Injection
5. Data Binding & View Binding
6. Retrofit
